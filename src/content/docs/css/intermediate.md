# CSS - Intermediate Concepts

Key topics for intermediate CSS include:

- Advanced Selectors:
    - Combinators: Descendant (` `), Child (`>`), Adjacent Sibling (`+`), General Sibling (`~`)
    - Pseudo-classes (e.g., `:hover`, `:focus`, `:nth-child()`, `:first-child`, `:last-child`, `:not()`)
    - Pseudo-elements (e.g., `::before`, `::after`, `::first-letter`, `::selection`)
- Box Model In-depth: `box-sizing: border-box`
- Positioning: `static`, `relative`, `absolute`, `fixed`, `sticky`, `z-index`
- Flexbox Layout:
    - Container properties (`display: flex`, `flex-direction`, `flex-wrap`, `justify-content`, `align-items`, `align-content`)
    - Item properties (`flex-grow`, `flex-shrink`, `flex-basis`, `order`, `align-self`)
- Grid Layout:
    - Container properties (`display: grid`, `grid-template-columns`, `grid-template-rows`, `grid-gap`, `justify-items`, `align-items`)
    - Item properties (`grid-column`, `grid-row`, `justify-self`, `align-self`)
- Responsive Design:
    - Media Queries (`@media`)
    - Mobile-first approach
- Transitions: `transition` property, `transition-property`, `transition-duration`, `transition-timing-function`, `transition-delay`
- Transforms: `transform` property (`translate()`, `rotate()`, `scale()`, `skew()`)
- Variables (Custom Properties): `var(--variable-name)`
- Basic Animations (`@keyframes`, `animation` property)
- Gradients (`linear-gradient()`, `radial-gradient()`)
- Box Shadows (`box-shadow`) and Text Shadows (`text-shadow`)
- Pseudo-elements for UI enhancements (`::before`, `::after`)
